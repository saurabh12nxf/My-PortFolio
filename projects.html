Traffic Light Simulator

Designed and developed an interactive Traffic Light Simulator featuring real-time message and image displays based on light color.

  • Real-Time Updates: Display messages and images corresponding to the traffic light color (Red, Yellow, Green).
  • Manual Control: Enable manual control of the traffic lights using buttons for enhanced user interaction.
  • User Interface: User-friendly interface for seamless operation and visual appeal.
  • Responsive Design: Adaptable to different screen sizes and resolutions.

Technologies Used: HTML & CSS, Javascript
